Símbolos, sintaxis, esqueleto de los programas:
Simbolos:
+ --> SUMA
- --> RESTA
* --> MULTIPLICACIÓN
/ --> DIVISIÓN
// --> DIVISION ENTERA
MOD --> RESIDUO
^ --> POTENCIA

Sintaxis:
Conjunto de caracteres válidos (máx. 128; ASCII 7 bits):
Mayúsculas [A-Z]
Minúsculas [a-z]
Dígitos [0-9]
Caracteres especiales: + - * / ^ < > ~ : ; . _ ? \ ( ) [ ]
Se distinguen imprimibles de no imprimibles.
Un programa PROLOG es una sucesión de términos:
Constantes.
Variables.
Estructuras (functores).

Esqueleto de programa:
:- initialization hello_world, halt.
hello_world :-
  write('Hello, World!'), nl.

Aplicación a descargar para editar y ejecutar:
Necesitas descargar el SWI-Prolog en https://www.swi-prolog.org/download/stable.

A que tipo de paradigma pertenece: Lógico y declarativo.

Un ejemplo de un programa escrito:

Factorial:

factorial(N,F) :-
N>0,
N1 is N-1,
factorial(N1,F1),
F is N * F1.
